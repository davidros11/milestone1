## instructions
Compile with

```bash
g++ -std=c++14 *.cpp -Wall -Wextra -Wshadow -Wnon-virtual-dtor -pedantic -o
a.out -pthread
```

Then run with

```bash
./a.out [text-file]
```

text file should be in the same foldier as the source code.
